# storybook-vue-example

Use Poi to develop and build your storybook playground.

## Folder structure

- `src/Button.vue`: The component you want to demonstrate.
- `src/stories/index.js`: The stories for your component
- `.storybook/config`: Storybook config file
- `.storybook/addons`: Storybook addons (optional)
- `.storybook/poi.config.js`: Poi config file

## npm scripts

- `yarn storybook`: Launch dev server
- `yarn storybook:build`: Build your storybook playground

Have fun!
